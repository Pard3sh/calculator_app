## How to Run (Python3)
```bash
# clone the repository
# change directory to the project
cd calculator_app 

# create a virtual environment and activate it 
python3 -m venv venv
source venv/bin/activate

# install from requirements
pip3 install -r requirements.txt

# run entry point
python3 main.py
```
